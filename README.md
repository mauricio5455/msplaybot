# MS PLAY BOT

Um simples bot de WhatsApp que roda em nodejs pensado para funcionar no Termux, mas também é compatível com o ambiente Windows e Linux.

## 📦 Dependências
Para funcionar corretamente, o bot precisa do FFMPEG, e GIT instalados; além do NodeJs, claro.

### Termux
```bash
pkg i nodejs
pkg i ffmpeg
pkg i git
```

### Windows / Linux

- [NodeJs](https://nodejs.org/pt-br)
- [FFMPEG](https://www.ffmpeg.org/)
- [GIT](https://git-scm.com/)

## Instação e Execução

Com as dependências instaladas, basta clonar o projeto, navegar até a sua pasta e executar node start.

- Clonar projeto

```bash
 git clone https://github.com/mauricio5455/msplaybot.git
```

- Iniciar aplicação
```bash
cd msbot
npm i
npm start
```
